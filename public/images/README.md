# 圖片放置說明

## 建議命名規則：

-   臺北市：`taipei-1.jpg`, `taipei-2.jpg`, ...
-   新北市：`newtaipei-1.jpg`, `newtaipei-2.jpg`, ...
-   臺中市：`taichung-1.jpg`, `taichung-2.jpg`, ...
-   高雄市：`kaohsiung-1.jpg`, `kaohsiung-2.jpg`, ...
